## 개념

- 네트워크 연결된 모든 장치들
- 데이터를 송수신하거나 중계할 수 있는 장치들
- IP주소를 가지고 있는 모든 장치들이 노드이다.
    - 컴퓨터 서버 라우터 스위치 등등

## 종류

- 호스트 노드
    - 데이터를 직접 송/수신 하는 노드 장치
        - 컴퓨터,서버
- 중간 노드
    - 중간에서 데이터를 중계해주는 노드 장치
        - 라우터, 스위치
- 엔드 노드
    - 데이터가 출발하거나 도착하는 노드 장치
        - 사용자 단말기
- 라우팅 노드
    - 경로를 설정해주고 데이터를 전달
        - 라우터,게이트웨이

## 연결 관계

노드들은 링크로 연결되어 있다.

노드 + 링크를 통해서 네트워크 토폴로지를 형성할 수 있다.

성형 구조 등등이 있음

## 노드의 식별 - IP 주소

노드는 IP 주소로 식별이 가능하다.

IP는 노드의 위치를 알려주는 역할

MAC은 노드 안의 물리적 주소를 알려주는 역할을 한다.

## 노드와 라우터·스위치의 차이

| 구분 | 역할 | OSI 계층 | 비고 |
| --- | --- | --- | --- |
| **스위치(Switch)** | 같은 네트워크 내 장비 간 데이터 전송 | 2계층 (데이터링크) | MAC 기반 |
| **라우터(Router)** | 네트워크 간 데이터 전달, 경로 결정 | 3계층 (네트워크) | IP 기반 |
| **노드(Node)** | 모든 네트워크 참여 장비의 총칭 | 모든 계층 포함 가능 | 포괄 개념 |
